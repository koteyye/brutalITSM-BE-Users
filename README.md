## User-service

BE для управление пользователями и функционал авторизации в https://github.com/koteyye/brutalITSM

## REST Methods
* GET SignIn - Авторизация;
* GET me - Получить данные по текущему пользоватьлю;
* GET getUsers - Получить список пользователей;
* GET getUser - Получить пользователя по ID;
* DEL deleteUser - Удалить пользователя по ID;
* POST createUser - Создать пользователя;
